## 4. Property Multiplication Through Reverse Financing

### What is Reverse Financing?

Reverse financing is using your existing property equity to acquire more properties, creating a snowball effect of wealth accumulation.

### Strategy 1: Loan Against Property (LAP)

**How it works:**
```
Existing Property Value: ‚Çπ1 Cr
LTV (Loan-to-Value): 60%
Available Loan: ‚Çπ60 Lakhs
Use: Down payment for 2-3 new properties
```

**Interest Rates:** 8.5-11% (higher than home loans)

**Example: One Big Property ‚Üí Multiple Small Properties**

| Starting Position | After Reverse Financing |
|-------------------|------------------------|
| 1 Property @ ‚Çπ1.35 Cr (Sarjapur) | |
| Equity: ‚Çπ80L | |
| Monthly Rent: ‚Çπ38,000 | |
| | **After LAP (‚Çπ80L @ 60% = ‚Çπ48L):** |
| | Property 2 (Devanahalli): ‚Çπ75L (Down: ‚Çπ15L) |
| | Property 3 (Devanahalli): ‚Çπ80L (Down: ‚Çπ16L) |
| | Cash Reserve: ‚Çπ17L |
| | Total Rent: ‚Çπ38K + ‚Çπ20K + ‚Çπ22K = ‚Çπ80,000 |

### Strategy 2: Top-Up Home Loan

**Advantages over LAP:**
- Lower interest rate (same as home loan: 8.5-9%)
- Tax benefits available
- Longer tenure possible

**Eligibility:**
- Original loan tenure completed > 1 year
- Good repayment track record
- Property value appreciation

### Strategy 3: Balance Transfer + Top-Up

**How it works:**
1. Transfer existing loan to bank offering lower rate
2. Get top-up based on appreciated property value
3. Use top-up as down payment for new property

**Example:**

```
Original Loan: ‚Çπ50L @ 10% with Bank A
Property Current Value: ‚Çπ85L

Transfer to Bank B:
- New Rate: 8.75%
- EMI Savings: ‚Çπ3,500/month
- Top-Up Available: ‚Çπ20L (based on appreciation)
- Use ‚Çπ20L for new property down payment
```

### Strategy 4: The BRRRR Method (Adapted for India)

**B**uy ‚Üí **R**enovate ‚Üí **R**ent ‚Üí **R**efinance ‚Üí **R**epeat

| Step | Action | Timeline |
|------|--------|----------|
| Buy | Purchase undervalued property | Month 1 |
| Renovate | Add 20-30% value through improvements | Month 2-4 |
| Rent | Achieve market rent | Month 5 |
| Refinance | Extract equity through LAP | Month 12 |
| Repeat | Use extracted capital for next property | Month 13 |

## 6. Tax Liability & Saving Options

### Understanding Property Taxes in India

#### A. Taxes During Purchase

| Tax/Charge | Rate | Paid By |
|------------|------|---------|
| Stamp Duty (Karnataka) | 5% (3% for women) | Buyer |
| Registration Fee | 1% | Buyer |
| GST (Under Construction) | 5% (without ITC) | Buyer |
| TDS (If seller is NRI) | 20% + surcharge | Buyer deducts |

**Saving Tip:** Register in woman's name ‚Üí Save 2% stamp duty on ‚Çπ1Cr = ‚Çπ2 Lakhs

#### B. Taxes During Holding Period

**1. Rental Income Tax**

```
Taxable Rental Income = Gross Rent - Standard Deduction (30%) - Municipal Taxes - Interest on Home Loan

Example:
Gross Annual Rent: ‚Çπ3,60,000
Standard Deduction (30%): -‚Çπ1,08,000
Municipal Taxes: -‚Çπ12,000
Home Loan Interest: -‚Çπ4,00,000
Taxable Income: -‚Çπ1,60,000 (Loss!)

This loss can be set off against other income (up to ‚Çπ2L/year)
```

**2. Property Tax**
- Paid to BBMP/Local body
- Rate: 0.2-0.5% of guidance value
- Due: April each year

#### C. Taxes on Sale (Capital Gains)

| Holding Period | Tax Type | Rate |
|----------------|----------|------|
| < 2 years | Short-Term Capital Gains | As per income slab (up to 30%) |
| > 2 years | Long-Term Capital Gains | 12.5% (without indexation) |

**LTCG Calculation (Post-2024 Budget):**

```
Sale Price: ‚Çπ1,50,00,000
Purchase Price: ‚Çπ80,00,000
Capital Gain: ‚Çπ70,00,000
LTCG Tax (12.5%): ‚Çπ8,75,000
```
